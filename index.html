There are three ways of inserting a style sheet:

  1. Inline css
  2. Internal css
  3. External css

  1. Inline CSS

    2. Inline styles are defined within the "style" attribute of the relevant element:

    Syntax

    <p style="color:red;">This is a paragraph.</p>

  2. Internal css

    3. The internal style is defined inside the <style> element, inside the head section.

    Syntax

    <head>
    <style>
    body { background-color: linen; }
    h1 { color: maroon; margin-left: 40px; }
    </style>
    </head>

  3. External

    4. Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section.

    Syntax

    <link rel="stylesheet" href="mystyle.css">
