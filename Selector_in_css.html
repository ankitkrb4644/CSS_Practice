CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Syntax

<style>

// Element selector:

div{ background-color: antiquewhite; }

// class Selector

.red{
color: rgb(12, 13, 13); background-color: rgb(245, 214, 214);
}

// Id Selector

#color {
background-color:rgb(245, 214, 214) ;
}
